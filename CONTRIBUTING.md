# Contributing to MacStats

Thank you for your interest in contributing! We welcome bug reports, feature requests, and pull requests.

## How to Contribute

1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  **Create a branch** for your feature or fix: `git checkout -b feature/awesome-feature`
4.  **Make your changes**. Ensure you follow the code style.
5.  **Commit** your changes: `git commit -m "Add awesome feature"`
6.  **Push** to your branch: `git push origin feature/awesome-feature`
7.  **Open a Pull Request**.

## Code Style

We use `SwiftLint` to maintain code quality. Please ensure you have it installed and run `swiftlint` before committing.

## Reporting Bugs

Please use the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md) when opening an issue.
